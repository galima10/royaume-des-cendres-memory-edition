# Le Royaume des Cendres - Memory Edition
Issu du jeu original que j'ai développé durant une Game Jam, Le Royaume des Cendres - Memomry Edition reprend les codes du jeu original en mélangeant le gameplay du célèbre jeu du Memory.<br /><br /><br />
**Qu'est-ce que Le Royaume des Cendres ?**<br />
Élu meilleur univers sonore de la Game Jam MyDigitalSchool 2026, vous incarnez 2 héros au choix pour partir dans une aventure épique riche en épreuves.<br /><br />

*Synopsis :*<br />
Dans un univers médiéval fantastique appelé Le Royaume des Cendres, Clint et Astride affrontent la fureur d’un roi jaloux qui perçoit leur simple existence comme une menace à son règne.<br />

Durant le jeu, les héros se déplacent de vallée en vallée sur une carte affichée en vue top-down, ils devront affronter des ennemis qui leurs bloqueront le chemin, chaque combat se déroule en vue de profil. Le royaume est peuplé de créatures hostiles et recèle de nombreux secrets, permettant aux héros de découvrir des objets utiles à leur progression ou des armes améliorant leurs capacités. L’exploration et la découverte de secrets constituent le cœur de l’expérience de jeu.

## Comment y jouer ?
### Installation
Pour jouer au jeu Le Royaume des Cendres - Memory Edition, il faut :
- Télécharger le code source
- L'installer dans un serveur (WAMP, MAMP, XAMP)
- Lancer le serveur et le jeu !
### Mode d'emploi
Plusieurs ennemies se dressent sur votre passage.<br />
Battez-les pour arriver au bout de votre aventure !<br /><br />
Un plateau de 16 cartes sont disposés.<br />
Trouvez la paire d'attaque pour infliger des dégâts aux ennemies.<br /><br />
Attention !<br />
Si vous tombez sur une carte de l'ennemi, c'est lui qui vous infligera des dégâts !<br /><br />
Des paires objets sont également présentes dans le plateau !
- Cœur : Si vous trouvez cette paire, vous récupérez un point de vie
- Torche : Si vous trouvez cette paire, la paire d'attaque se transforme en paire d'attaque spéciale pour infliger plus de dégâts si trouvée
<br /><br />

Lorsque vous trouvez une paire d'attaque, le plateau se mélange à nouveau. Les objets sont donc à nouveau trouvables.<br />
Attention ! Cela signifie aussi que les cartes ennemis sont également mélangées à nouveau avec les autres cartes.<br /><br />


Précision : 
- Les cartes ennemis sont les cartes de l'ennemi du niveau actuel.
- Faites attention ! L'ennemi change à chaque niveau, donc les cartes ennemis aussi ! Ne confondez pas les cartes de l'ennemi du niveau actuel et les autres cartes !<br />
### Nombre de cartes et composition du plateau
- Une paire d'attaque
- Une paire de cœur
- Une paire de torche
- 2 cartes ennemis
- D'autres cartes seules qui remplissent le reste du plateau
